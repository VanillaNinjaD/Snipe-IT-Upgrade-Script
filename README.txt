THIS PULLS THE LATEST VERSION OF SNIPE-IT FROM MASTER, CREATES A BACKUP OF THE CURRENT SNIPE-IT WEB DIRECTORY, AND INSTALLS MASTER TO THE PRE-EXISTING WEB DIRECTORY.

This script was created for a very specific, personal purpose. Feel free to take and modify this script at your will.

The script was originally created on the following setup and only works if you have a running instance of Snipe-IT v3.0 or beyond

OS: Ubuntu 14.04 LTS Server
MYSQL: 5.5.50-0ubuntu0.14.04.1 (Ubuntu)
Apache: Apache/2.4.7 (Ubuntu)
PHP: PHP 5.5.9-1ubuntu4.19
SNIPE-IT: v3.0
